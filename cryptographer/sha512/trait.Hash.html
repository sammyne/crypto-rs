pub trait Hash: Write {
    fn block_size() -> usize;

    fn reset(&mut self);

    fn size() -> usize;

    fn sum(&mut self) -> Vec<u8>;
}

The Hash trait specifies the common interface for hash functions.

The write from Write trait adds more data to the running hash. It never returns an error.

Required methods

fn block_size() -> usize

block_size returns the hash's underlying block size. The Write method must be able to accept any amount of data, but it may operate more efficiently if all writes are a multiple of the block size.

fn reset(&mut self)

reset resets the Hash to its initial state

fn size() -> usize

size returns the number of bytes Sum will return

fn sum(&mut self) -> Vec<u8>

sum returns the resulting slice. It does not change the underlying hash state.
