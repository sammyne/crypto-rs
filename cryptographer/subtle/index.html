Functions

constant_time_byte_eq

constant_time_byte_eq returns 1 if x == y and 0 otherwise.

constant_time_compare

constant_time_compare returns 1 if the two slices, x and y, have equal contents and 0 otherwise. The time taken is a function of the length of the slices and is independent of the contents.

constant_time_copy

constant_time_copy copies the contents of y into x (a slice of equal length) if v == 1. If v == 0, x is left unchanged. Its behavior is undefined if v takes any other value.

constant_time_eq

constant_time_eq returns 1 if x == y and 0 otherwise.

constant_time_less_or_eq

constant_time_less_or_eq returns 1 if x <= y and 0 otherwise. Its behavior is undefined if x or y are negative or > 2**31 - 1.

constant_time_select

constant_time_select returns x if v == 1 and y if v == 0. Its behavior is undefined if v takes any other value.
