Modules

aes
module aes implements AES encryption (formerly Rijndael), as defined in U.S. Federal Information Processing Standards Publication 197
cipher
module cipher implements standard block cipher modes that can be wrapped around low-level block cipher implementations. See https://csrc.nist.gov/groups/ST/toolkit/BCM/current_modes.html and NIST Special Publication 800-38A
ed25519
module ed25519 implements the Ed25519 signature algorithm. See https://ed25519.cr.yp.to/
elliptic
module elliptic implements several standard elliptic curves over prime fields
hmac
module hmac implements the Keyed-Hash Message Authentication Code (HMAC) as defined in U.S. Federal Information Processing Standards Publication 198. An HMAC is a cryptographic hash that uses a key to sign a message. The receiver verifies the hash by recomputing it using the same key.
md5
module md5 implements the MD5 hash algorithm as defined in RFC 1321.
rand
rc4
module rc4 implements RC4 encryption, as defined in Bruce Schneier’s Applied Cryptography.
sha1
module sha1 implements the SHA-1 hash algorithm as defined in RFC 3174.
sha256
sha512
subtle
module subtle implements functions that are often useful in cryptographic code but require careful thought to use correctly.
x
module x implements supplementary cryptography libraries.
